## Description 
Streamify is a  real-time video calling web application built with the MERN stack (MongoDB, Express.js, React, Node.js). It features secure user authentication, one-on-one video calls, instant messaging with typing indicators, message threading, emoji reactions, and moreâ€”delivering an interactive and seamless communication experience similar to modern chat platforms.

# How to start this Streamify

1. Install node_modules inside Frontend & Backend directory using npm install command.
2. Ask for .env file.
3. Start Backend server using npm start command.
4. Start Frondend Server using npm run dev command.
